# 2do_parcial
2do parcial (Taller de aplicaciones de tecnologías en internet)

1. ¿Cómo lo implementamos?

El proyecto lo realizamos en equipo con Jheny y Jorge. Usamos GitHub para trabajar de forma organizada y compartir el código entre todos. Cada integrante podía avanzar en una parte del proyecto y luego unir los cambios al trabajo principal.

Desarrollamos una aplicación web utilizando HTML para la parte visual y Python para la lógica del sistema. También implementamos una base de datos para guardar la información del sistema. Dentro de la base de datos creamos varias tablas para organizar los datos: ventas, detalles de venta, laboratorios, usuarios, clientes y medicamentos.

Durante el desarrollo tratamos de mantener una buena organización del proyecto para que todos podamos entender el código y trabajar de forma ordenada.

2. ¿Qué aprendimos?

En este proyecto aprendimos principalmente a trabajar en equipo usando GitHub, entendiendo cómo compartir cambios y combinar el trabajo de todos.

También comprendimos mejor cómo funciona una aplicación web completa, conectando la parte visual con la lógica del sistema y usando una base de datos para guardar información.

En general, el proyecto nos ayudó a entender mejor el proceso de desarrollo de una aplicación, desde la idea inicial hasta tener un sistema funcional.
